# Web Scraping Projects in Python
![Web scraping Illustration](./webscrapping.png)

This repository contains a collection of Python-based web scraping projects implemented in Jupyter notebooks. These projects demonstrate how to extract and analyze data from various websites using Python libraries like `requests`, `BeautifulSoup`, and APIs.

## Projects

### 1. **Automating Crypto Website API**
- **Description**: This project uses an API to fetch real-time cryptocurrency data (e.g., prices, market cap) from a crypto website and processes it for analysis.
- **Notebook**: [Python_Automating_Crypto_Website_API.ipynb](https://github.com/maxim-eyengue/Python-Codes/blob/main/Web_Scraping_Projects/API/Python_Automating_Crypto_Website_API.ipynb)
- **Key Features**:
  - Fetches data from a cryptocurrency API.
  - Processes and displays real-time crypto metrics.
- **Libraries Used**: `requests`, `pandas`.

---

### 2. **Web Scraping Amazon**
- **Description**: This project scrapes product data (e.g., name, price, rating) from Amazon's website and saves it into a structured format for further analysis.
- **Notebook**: [Python_Web_Scraping_Amazon.ipynb](https://github.com/maxim-eyengue/Python-Codes/blob/main/Web_Scraping_Projects/Amazon/Python_Web_Scraping_Amazon.ipynb)
- **Key Features**:
  - Extracts product details from Amazon search results.
  - Saves data into a CSV file for easy analysis.
- **Libraries Used**: `BeautifulSoup`, `requests`, `pandas`.

---

### 3. **Web Scraping Companies from Wikipedia**
- **Description**: This project scrapes data about companies listed on Wikipedia, such as their names, industries, and founding years, and organizes it into a structured format.
- **Notebook**: [Python_Web_Scraping_Companies.ipynb](https://github.com/maxim-eyengue/Python-Codes/blob/main/Web_Scraping_Projects/Wikipedia/Python_Web_Scraping_Companies.ipynb)
- **Key Features**:
  - Extracts company data from Wikipedia tables.
  - Saves the data into a CSV file for further use.
- **Libraries Used**: `BeautifulSoup`, `requests`, `pandas`.

---

## How to Use
1. Clone this repository or download the specific notebook you need.
2. Open the notebook in Jupyter or any compatible environment.
3. Follow the instructions provided in the notebook to run the web scraping scripts.

## Requirements
- Python 3.x
- Jupyter Notebook
- Libraries: `requests`, `BeautifulSoup`, `pandas`.
